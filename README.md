# Educational Technology Portfolio
Developed a portfolio demonstrating the use of digital tools, collaboration platforms, creativity, and design thinking to enhance learning and problem-solving in educational and professional contexts. Applied a wide range of technologies to create interactive, innovative, and user-focused solutions.

This is a professional portfolio website built using HTML, CSS, and JavaScript. It highlights academic background, technical skills, and projects related to business, finance, and data analytics.

- Responsive single-page layout
- Smooth scrolling navigation
- Project showcase section
- Skills and education summary

## Technologies Used

- CSS3
- JavaScript

## Purpose
Designed for internship applications, LinkedIn portfolio support, and showcasing academic/business analytics projects.
